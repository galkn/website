Etsy DataFindr

I worked on the core infrastructure team at Etsy, where I lead the design and development of Etsy DataFindr, an API-endpoint discovery and documentation front-end app with an interface to securely and asynchronously run API queries on Etsy's database and view JSON output. New updates to the API were made to automatically reflect in the app. DataFindr is used by Etsy front-end and mobile application developers and recently discussed on Etsy's engineering blog.
